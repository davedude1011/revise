Operators in Python: An Introduction
Operators in Python are symbols or keywords that perform operations on variables and values. Let's explore some of the basic operators:
1. Arithmetic Operators
These operators perform mathematical operations like addition, subtraction, multiplication, and division.
- Addition (+)
Example:
result = 5 + 3 # result will be 8
- Subtraction (-)
Example:
result = 10 - 4 # result will be 6
- Multiplication (*)
Example:
result = 2 * 3 # result will be 6
- Division (/)
Example:
result = 15 / 3 # result will be 5.0 (float)
2. Comparison Operators
These operators compare two values and return a Boolean result (True or False).
- Equal to (==)
Example:
is_equal = 5 == 5 # is_equal will be True
- Not equal to (!=)
Example:
not_equal = 10 != 5 # not_equal will be True
- Greater than (>)
Example:
greater_than = 15 > 10 # greater_than will be True
- Less than (<)
Example:
less_than = 7 < 9 # less_than will be True
3. Logical Operators
These operators combine multiple conditions and return a Boolean result.
- AND (and)
Example:
both_conditions = (5 > 3) and (10 < 15) # both_conditions will be True
- OR (or)
Example:
either_condition = (5 < 3) or (10 < 15) # either_condition will be True
- NOT (not)
Example:
not_condition = not (5 > 3) # not_condition will be False
4. Assignment Operators
These operators assign values to variables.
- Assignment (=)
Example:
x = 10 # x is assigned the value 10
- Add and Assign (+=)
Example:
y = 5
y += 3 # y will be 8
Conclusion
Understanding operators is crucial for writing effective Python code. They allow you to manipulate data, make decisions, and control the flow of your programs.
