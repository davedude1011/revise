Loops in Python: For and While
Loops are fundamental constructs in Python that enable you to repeat a block of code multiple times. There are two primary types of loops: for and while.
For Loop
A for loop is used for iterating over a sequence (that is either a list, tuple, dictionary, string, or other iterable objects). The basic syntax is as follows:
for variable in sequence:
ㅤ# code to be executed
Here's an example that prints each element of a list:
numbers = [1, 2, 3, 4, 5]
for num in numbers:
ㅤ print(num)
While Loop
A while loop repeats a block of code as long as a specified condition is true. The basic syntax is:
while condition:
ㅤ# code to be executed
Here's an example that prints numbers from 1 to 5 using a while loop:
count = 1
while count <= 5:
ㅤ print(count)
ㅤ count += 1
Conclusion
Understanding how to use for and while loops is crucial for writing efficient and concise code in Python. Loops are powerful tools that allow you to automate repetitive tasks and process data more effectively.
